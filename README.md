#  Library Management System

[![Tech Stack](https://img.shields.io/badge/Tech-C%23%20%7C%20ASP.NET%20Web%20Forms%20%7C%20SQL%20Server-orange)](https://dotnet.microsoft.com/en-us/apps/aspnet/web-apps/web-forms)  
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

A modern web application to manage library operations efficiently. Admins can manage books, members, and issued books, while the dashboard provides insightful charts for quick analysis.

---

##  Features
- Add, edit, and view **Books** and **Members**
- Issue and return books with status tracking
- Category-wise book management
- Interactive **Dashboard** with bar, pie, and line charts
- Admin authentication and secure logout

---

##  Tech Stack
- **Frontend:** ASP.NET Web Forms, Bootstrap 5, Chart.js  
- **Backend:** C#  
- **Database:** SQL Server
